Thanks for checking out fCraft.

fCraft requires Microsoft .NET Framework 3.5+ (on Windows) or Mono 2.6.4+ (on Linux / Unix / Mac OS X)

Installation instructions: https://sourceforge.net/apps/mediawiki/fcraft/index.php?title=Installation

For a full list of commands, type "/help commands all" in game or console.

For help/support, join #fCraft channel on Esper.net IRC.
